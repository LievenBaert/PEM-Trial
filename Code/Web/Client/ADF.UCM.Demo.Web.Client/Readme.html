ArcelorMittal Angular Client Template

Getting started

  1. Read the wiki to start with the ArcelorMittal Angular Template
  2. Install Node
  3. In the project folder command prompt, run npm install -g gulp bower typings
  4. Install the build tools by running npm install
  5. Install the client-side packages by running bower install
  6. Install the typescript type definitions by running typings install --global
  7. Before starting development run npm run dev to start Browsersync and watching for file changes

Proxy Requirements

This template is supported by an private package manager hosted at packagefeeds.sidmar.be
Because of this we don't need to configure a proxy on your development machine.
In cause you're are having problems with package managers other then npm,bower or nuget, you might need to setup your own.

  1. Install Fiddler as a proxy for npm & bower packages
